English Puzzle Game

Hệ sinh thái học từ vựng kết hợp Gamification. Nổi bật với cơ chế giải đố ma trận chữ, hệ thống ôn tập Spaced Repetition và công cụ CMS kiểm thử dữ liệu tự động.

Flutter Provider Python (Flask) Vanilla JS Gamification
Repo

English Puzzle Game - Hệ Sinh Thái Học Tập Đa Nền Tảng

Mô tả tổng quan: Không chỉ là một ứng dụng di động, đây là sự kết hợp giữa cơ chế giải đố gây nghiện và phương pháp giáo dục hiện đại. Dự án thể hiện tư duy phát triển sản phẩm toàn diện: từ thiết kế luồng người dùng (UX), quản lý trạng thái (State Management) đến xây dựng công cụ quản trị dữ liệu (CMS) độc lập.

Điểm nhấn Công nghệ & Quá trình Phát triển

  • Kiến trúc & UX/UI (Flutter): Quản lý luồng dữ liệu phức tạp mượt mà bằng Provider. Ứng dụng kỹ thuật thiết kế "Chameleon Theme" (Tắc kè hoa), tự động tái cấu trúc toàn bộ mã màu, gradient và shadow theo chủ đề hệ thống.
  • Cơ chế Gamification & Spaced Repetition: Xây dựng hệ thống kinh tế trong game (Vàng, Gợi ý, Chuỗi Journey). Tích hợp module luyện tập thông minh tự động mở khóa dựa trên tiến độ, hỗ trợ 3 phương pháp tiếp cận: Trắc nghiệm, Nghe (tích hợp Text-to-Speech) và Viết.
  • Công cụ CMS & Data Pipeline (Python/Flask): Tự xây dựng một "Data Dashboard Pro Max" trên nền tảng Web. Tool giúp trực quan hóa ma trận từ vựng, bắt lỗi logic mapping, và cung cấp môi trường rà soát ngữ nghĩa (QA) để xuất trực tiếp file Database chuẩn hóa.
  • Hiệu suất (Optimization): Tối ưu thuật toán tìm kiếm và nội suy mảnh ghép. Đảm bảo ứng dụng hoạt động 100% Offline với bộ Dataset khổng lồ mà không tiêu hao nhiều tài nguyên thiết bị.

Smart Weather App

Ứng dụng tra cứu thời tiết linh hoạt theo giờ, tích hợp bản đồ Geocoding và API định vị GPS tự động tối ưu UX.

Vanilla JS RESTful API Geolocation
Code Demo

Smart Weather App (Dự Báo Thời Tiết)

Mô tả tổng quan: Một ứng dụng web cung cấp thông tin dự báo thời tiết theo giờ chi tiết, nhanh chóng và chính xác được viết hoàn toàn bằng HTML, CSS và JavaScript thuần.

Kỹ thuật xử lý & Nổi bật

  • API Integration: Kết hợp khéo léo 2 API. Sử dụng Nominatim OpenStreetMap API để chuyển đổi địa danh tiếng Việt thành tọa độ, sau đó gọi OpenWeatherMap API để lấy dữ liệu thời tiết.
  • Geolocation: Tích hợp API định vị toàn cầu của trình duyệt để tự động lấy vị trí hiện tại của người dùng chỉ với 1 click chuột.
  • UX Optimization: Xử lý bất đồng bộ (Fetch/Promises) kết hợp với các hiệu ứng báo trạng thái Loading, báo lỗi rõ ràng khi không tìm thấy địa danh. Hỗ trợ tìm kiếm nhanh bằng phím Enter.

Centralized Authentication System (FreeRADIUS)

Mô tả tổng quan: Dự án yêu cầu thiết lập một máy chủ quản lý xác thực tập trung (AAA) bảo mật cho mạng nội bộ, sử dụng phần mềm nguồn mở FreeRADIUS trên nền hệ điều hành Linux CentOS.

Vai trò (Leader) & Nhiệm vụ

  • Lên kế hoạch, thiết kế kiến trúc mạng tổng thể và phân chia tiến độ công việc cho 3 thành viên khác trong nhóm.
  • Trực tiếp cài đặt và cấu hình Core FreeRADIUS service, tạo lập user/group policy trên CentOS.
  • Thiết lập kết nối an toàn giữa các thiết bị mạng (Switch/Access Point) và Radius Server.
  • Kiểm tra, gỡ lỗi (troubleshoot) quá trình xác thực và tổng hợp tài liệu báo cáo nghiệm thu.
